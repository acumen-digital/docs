# De-index user

This endpoint allows you De-index a user in the event that a user deactivates their account on your platform.

### Endpoint

<Box
    color='white'
    bg='red'
    p={0}
    my={1}
    fontWeight={"bold"}
    width={62}
    borderRdius={3}
>DELETE
</Box>

```
https://api.thepeer.co/users/delete/{{USER_REFERENCE}}
```

|URL Parameter|Description|
|---|---|
|`{{USER_REFERENCE}}`|the `reference` returned when the user was indexed|

### Sample response

```json
{
  "message": "user de-indexed"
}
```
