# Events

## Send Events

## Events from Thepeer

|Event name|Description|
|---|---|
|`send.success`|Send transaction was sucessful|
|`send.insufficient_funds`|Business partner does not have sufficient wallet value to honor the transaction
|`send.server_erorr`|A error from Thepeer|
|`send.invalid_receipt`|Invalid receipt|
